## ğŸ”§ Integration Options

<Tabs items={['HTTP Client', 'Generated SDKs', 'cURL Examples']}>
<Tab value="HTTP Client">
```javascript
// JavaScript/Node.js example
const response = await fetch('https://api2.rhombussystems.com/api/endpoint', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-auth-scheme': 'api-token',
    'x-auth-apikey': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    // Your request data
  })
});

const data = await response.json();
```
</Tab>
<Tab value="Generated SDKs">
```bash
# Generate client libraries using OpenAPI Generator
openapi-generator-cli generate \
  -i https://api2.rhombussystems.com/api/openapi/public.json \
  -g python \
  -o ./rhombus-python-client

# Or use other generators: java, csharp, go, php, etc.
```
</Tab>
<Tab value="cURL Examples">
```bash
# Basic API request with token authentication
curl -X POST \
  -H "Content-Type: application/json" \
  -H "x-auth-scheme: api-token" \
  -H "x-auth-apikey: YOUR_API_KEY" \
  -d '{"key": "value"}' \
  https://api2.rhombussystems.com/api/endpoint
```
</Tab>
</Tabs>
