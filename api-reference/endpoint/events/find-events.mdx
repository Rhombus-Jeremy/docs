---
title: 'Find Events by Location'
description: 'Retrieve events filtered by location and time parameters'
openapi: 'POST /api/event/findEventsByLocationAndTime'
---

Retrieve events from specific locations within a time range. This endpoint is essential for incident investigation, compliance reporting, and security analysis.

## Event Types

The system captures various types of security events:

- **Access Events**: Door access, badge reads, access denials
- **Motion Detection**: Camera-based motion alerts
- **Policy Violations**: Rule-based security alerts
- **System Events**: Device status changes, connectivity issues
- **Manual Events**: User-created incident markers

## Filtering Options

<Tabs>
<Tab title="Time-based Filtering">
```json
{
  "startTime": "2024-01-01T00:00:00Z",
  "endTime": "2024-01-02T00:00:00Z",
  "timezone": "America/Los_Angeles"
}
```
</Tab>

<Tab title="Event Type Filtering">
```json
{
  "eventTypes": ["ACCESS_GRANTED", "ACCESS_DENIED", "MOTION_DETECTED"],
  "severity": ["HIGH", "CRITICAL"]
}
```
</Tab>

<Tab title="Location Filtering">
```json
{
  "locationUuids": ["loc-123", "loc-456"],
  "includeSubLocations": true
}
```
</Tab>
</Tabs>
