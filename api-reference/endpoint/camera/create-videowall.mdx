---
title: 'Create Video Wall'
description: 'Create a new video wall configuration for displaying multiple camera feeds'
openapi: 'POST /api/camera/createVideoWall'
---

Create a video wall configuration to display multiple camera feeds simultaneously. Video walls are ideal for security operations centers, monitoring stations, and public displays.

## Video Wall Features

- **Multi-camera Display**: Show multiple feeds in grid layouts
- **Flexible Layouts**: Support for various grid configurations (2x2, 3x3, 4x4, custom)
- **Dynamic Content**: Automatically cycle through camera feeds
- **Alert Integration**: Highlight cameras with active alerts
- **Remote Management**: Configure and control video walls remotely

## Layout Options

<Tabs>
<Tab title="Grid Layout">
```json
{
  "layout": "grid",
  "rows": 3,
  "columns": 3,
  "cameras": [
    "camera-uuid-1",
    "camera-uuid-2",
    "camera-uuid-3"
  ]
}
```
</Tab>

<Tab title="Focus Layout">
```json
{
  "layout": "focus",
  "primaryCamera": "camera-uuid-1",
  "secondaryCameras": [
    "camera-uuid-2",
    "camera-uuid-3",
    "camera-uuid-4"
  ]
}
```
</Tab>

<Tab title="Rotation Layout">
```json
{
  "layout": "rotation",
  "rotationInterval": 30,
  "cameras": [
    "camera-uuid-1",
    "camera-uuid-2",
    "camera-uuid-3"
  ]
}
```
</Tab>
</Tabs>

<Tip>
Video walls can be configured to automatically switch to alert cameras when incidents occur, ensuring security teams never miss critical events.
</Tip>
