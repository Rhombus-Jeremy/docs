# Rhombus Zapier Integration

<Info>
**No-Code Automation Platform**

Zapier is the leader in workplace automation, empowering organizations to automate workflows. When integrated with Rhombus, it streamlines daily tasks by connecting the Rhombus Rules Engine to any app in minutes.

Connect Rhombus with thousands of popular apps through our native Zapier integration. Automate your security workflows, notifications, and responses without writing a single line of code.

## What is Zapier?

Zapier lets you connect Rhombus with 8,000+ other apps to automate repetitive tasks. Create "Zaps" (automated workflows) that trigger actions in other apps when events occur in your Rhombus system.

## Available Triggers

Rhombus provides several triggers that can start your automated workflows:

### New Audit Record
Triggers when a new audit record for a specified event is created in your Rhombus system.

**Use Cases:**
- Log security events to spreadsheets
- Send notifications for specific activities
- Create tickets in project management tools

### New Button Event
Triggers when a Rhombus button device is pressed.

**Use Cases:**
- Send emergency SMS alerts
- Notify security teams via Slack
- Trigger automated responses

### New Clip Created
Triggers when a new video clip is saved in your Rhombus system.

**Use Cases:**
- Backup important clips to cloud storage
- Send clip notifications to stakeholders
- Analyze footage with AI services

### New Door Event
Triggers when door-related events occur (open, close, access attempts).

**Use Cases:**
- Track access patterns
- Send unauthorized access alerts
- Log entry/exit data

## Available Actions

Control your Rhombus system through these automated actions:

### Unlock Access Controlled Door
Remotely unlock doors in your Rhombus access control system.

**Use Cases:**
- Grant access when someone checks in via gym management software
- Unlock doors based on calendar events
- Provide temporary access through form submissions

### Reboot Camera
Restart Rhombus cameras remotely.

**Use Cases:**
- Schedule regular camera maintenance
- Restart cameras after network issues
- Perform bulk camera reboots

### Play Audio
Play audio messages through Rhombus audio gateways.

**Use Cases:**
- Broadcast announcements
- Play alerts during emergencies
- Provide audio feedback for events

## Getting Started

### Step 1: Set Up Your Zapier Account

1. Visit [zapier.com](https://zapier.com) and create an account
2. Search for "Rhombus" in the app directory
3. Connect your Rhombus account using your API credentials

### Step 2: Create Your First Zap

<Steps>
  <Step title="Choose a Trigger">
    Select a Rhombus trigger that matches your use case (e.g., "New Button Event")
  </Step>
  
  <Step title="Configure the Trigger">
    Set up the specific conditions for when the trigger should fire
  </Step>
  
  <Step title="Choose an Action">
    Select what should happen in another app when the trigger fires
  </Step>
  
  <Step title="Test and Activate">
    Test your Zap to ensure it works correctly, then turn it on
  </Step>
</Steps>

### Step 3: Authentication

You'll need your Rhombus API credentials to connect:

1. Log into the [Rhombus Console](https://console.rhombussystems.com/settings/api-management/)
2. Generate an API key if you don't have one
3. Use your API key to authenticate the Zapier connection

<Warning>
Keep your API credentials secure. They provide access to your Rhombus system and should be treated like passwords.
</Warning>

## Advanced Features

### Multi-Step Zaps

Create complex workflows with multiple actions:

```
Trigger: New Audit Record (Unauthorized Access)
├── Action 1: Send SMS alert to security
├── Action 2: Create Slack channel for incident
├── Action 3: Lock down additional doors
└── Action 4: Save incident details to database
```

### Conditional Logic

Use Zapier's Filter and Paths features to create smart workflows:

- **Filters:** Only trigger actions when specific conditions are met
- **Paths:** Route different events to different actions
- **Delays:** Add time delays between actions

### Webhooks Integration

For advanced users, combine Zapier with Rhombus webhooks for real-time data processing and custom integrations.

## Pricing & Limits

Zapier offers various pricing tiers based on your automation needs:

- **Free Plan:** 100 tasks/month, 5 Zaps
- **Starter Plan:** 750 tasks/month, 20 Zaps
- **Professional Plan:** 2,000+ tasks/month, unlimited Zaps
- **Enterprise Plans:** Custom limits and features

<Info>
Tasks are counted each time a Zap runs. A single trigger can execute multiple actions, but each action counts as one task.
</Info>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Authentication Errors">
    **Problem:** Zapier can't connect to Rhombus
    
    **Solutions:**
    - Verify API key is correct and active
    - Check API key permissions in Rhombus Console
    - Ensure API key hasn't expired
    - Try reconnecting the Rhombus app in Zapier
  </Accordion>
  
  <Accordion title="Trigger Not Firing">
    **Problem:** Zap isn't triggering when expected
    
    **Solutions:**
    - Check trigger configuration settings
    - Verify the event is actually occurring in Rhombus
    - Review Zap history for error messages
    - Test the trigger manually in Zapier
  </Accordion>
  
  <Accordion title="Action Failures">
    **Problem:** Actions aren't executing properly
    
    **Solutions:**
    - Check action app authentication
    - Verify required fields are populated
    - Review error messages in Zap history
    - Test actions individually
  </Accordion>
</AccordionGroup>
