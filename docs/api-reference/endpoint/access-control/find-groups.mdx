---
title: 'Find Access Control Groups'
description: 'Retrieve all access control groups in your organization'
openapi: 'POST /api/accesscontrol/findAccessControlGroupsByOrg'
---

Retrieve a complete list of all access control groups in your organization. This endpoint is essential for managing group-based permissions and understanding your current access control structure.

## Common Use Cases

- **Dashboard Display**: Show all groups in admin interfaces
- **User Assignment**: List available groups when assigning users
- **Audit Reports**: Generate reports on group structures
- **Bulk Operations**: Prepare for bulk group modifications

<Tip>
Use this endpoint to build dynamic user interfaces that display current group structures and allow administrators to manage permissions efficiently.
</Tip>
